# AtomCode 灵感展示墙

展示、浏览、互动并二次创作用 **AtomCode** 打造的应用与有趣程序。

- 🎨 作品墙：网格展示 + 搜索 / 标签 / 范围（全部·我的·我赞过）/ 排序
- 🌌 第一屏：改变世界的科技里程碑时间轴 —— 一个小人走过绳索吊桥，走过 2026 后坐飞船升空驶入「未来」
- 🔀 二次创作：一键衍生并记录传承链（原作 → … → 当前）
- ❤️ 互动：点赞、浏览量、评论
- ✏️ 作品管理：编辑 / 删除自己的作品、复制分享链接

纯前端，数据保存在浏览器本地（`localStorage`），无需后端即可运行。

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 深链参数（可选）

- `#work=<id>` —— 打开指定作品详情
- `?m=<N>` —— 第一屏定位到第 N 个里程碑（静止）
- `?start=<N>` —— 第一屏从第 N 站开始自动播放
